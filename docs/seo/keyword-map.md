## Keyword Rules

- Each important query should map to one primary page.
- Avoid making multiple pages compete for the same exact query.
- Use related phrases naturally in page copy, not as repeated lists.
- Prefer specific template pages for high-intent searches.
- Keep public safety rules above traffic goals.
