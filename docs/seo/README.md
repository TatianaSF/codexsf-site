# SEO Documentation

This folder contains the working SEO plan for the public Codex SF site.

## Files

- `seo-strategy.md` defines the overall search direction.
- `metadata-map.md` maps public routes to titles, descriptions, H1s, and search
  intent.
- `keyword-map.md` groups target keywords by audience and intent.
- `technical-seo-checklist.md` tracks implementation and publishing checks.
- `content-roadmap.md` defines the growth path from current routes to future
  search-focused pages.
- `search-console-indexing.md` defines the production Search Console workflow.
- `indexing-url-list.md` lists URLs to inspect after deployment.

## Recommended Workflow

1. Update `metadata-map.md` before changing page metadata.
2. Use `keyword-map.md` to avoid overlapping search intent between pages.
3. Check `technical-seo-checklist.md` before publishing.
4. Use `content-roadmap.md` to decide which new pages to build next.
5. Use `search-console-indexing.md` after deployment.
