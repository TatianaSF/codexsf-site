# TatianaSF Link QA Checklist

Use this checklist before publishing public site changes that mention TatianaSF.

## Required Text And Link Behavior

- Confirm all visible mentions say exactly `TatianaSF`.
- Confirm no old Latin-script variants remain in rendered pages.
- Confirm every `TatianaSF` mention links to `https://www.google.com/search?q=TatianaSF`.
- Confirm every `TatianaSF` link opens in a new tab/window.
- Confirm external link attributes are present where applicable:
  - `target="_blank"`
  - `rel="noopener noreferrer"`

## Pages And Surfaces

- Check Home page.
- Check About page.
- Check Footer.
- Check Header if `TatianaSF` appears there.
- Check mobile layout.
- Check Markdown-rendered pages.
- Check HackKit pages.
- Check Resources pages.
- Check Events pages.
- Check Community page.

## Public Safety

- Confirm no private ops content was modified.
- Confirm no secrets, contacts, budgets, private notes, speaker notes, partner notes, volunteer assignments, or internal event plans were added.
